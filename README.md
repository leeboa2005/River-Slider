## 자동 슬라이더 기능 🔁

자동으로 동작하는 슬라이더 이며, 마우스 오버, 장지 버튼을 누르면 멈추고 마우스가 해당 영역을 벗어나거나 플레이 버튼을 누르면 다시 애니메이션이 동작합니다.
